Xbox

Overview

Xbox Series X is the latest console released. At £479, it is around the same price as the PS5. It offers powerful SSD and faster loading times so you quicly load into games. With Xbox Game Pass, you get access to a large collection of games to download

It supports graphics up to 4K and 120 FPS and backwards compatible with original Xbox games

Specs

  • Processor: AMP Zen 2
  • Memory: 16GB RAM
  • Storage: 1TB SSD
  • Resolution: 4K 120 FPS
  • Cost: £479

Pros

  • Xbox game pass allows you to download games on launch
  • Fast loading times with quick resume
  • Backwards compatable games with Xbox 360 and Xbox One

Cons

  • Few UI changes
  • Not many exclusive games
  • Still uses the same controller
